# ISHIGAKI NIGHT（石垣島ナイト）

石垣島のナイトライフ情報メディアサイト。

## URL
https://ishigaki-night.vercel.app/

## 技術スタック
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- next-intl (ja / en / zh / ko)
- Supabase（準備中）
- Vercel

## 開発
```bash
npm run dev
```

## ディレクトリ
- `src/app/[locale]/` — ページ
- `src/components/` — UIコンポーネント
- `src/lib/` — データ・ロジック
- `src/messages/` — 翻訳ファイル
